Create a financial AI prediction web application with the exact same design, layout, UI style, colors, animations, and user experience as this website:

https://fintechaipredictor.lovable.app

Keep the same branding style, dashboard structure, cards, navigation, sections, icons, charts, typography, spacing, transitions, and overall user flow.

==================================================
MAIN IDEA
==================================================

Users should be able to analyze their financial behavior in two different ways:

Option 1
Upload a CSV transaction file.

Option 2
Enter transaction information manually through a guided questionnaire.

Both methods must produce the exact same dashboard and AI results.

No fake data, mock predictions, placeholder values, or hardcoded statistics should ever be shown.

UPLOAD CSV
==================================================

Add a drag-and-drop upload section.

Requirements:

• Display uploaded filename.
• Show number of rows.
• Show detected columns.

• Show processing animation while AI models are running.

==================================================

Allow users to manually enter their financial information instead of uploading a CSV.

The questionnaire must have the same premium design language as the rest of the website.

Use a multi-step wizard with smooth animations.

Include a progress indicator.

Validate all required fields.

Allow users to edit previous steps.

Auto-save answers while navigating.

Collect every feature required by the machine learning models.

Include fields such as:

Transaction Details
- Transaction Amount
- Transaction Date

- Transaction Location

Financial Information

Behavior Information
